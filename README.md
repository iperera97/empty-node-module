# empty-node-module

## Check Whether passing varibles are empty or not 
### The following values are considered to be empty:

* "" (an empty string)
* 0 (0 as an integer)
* 0.0 (0 as a float)
* "0" (0 as a string)
* NULL
* FALSE
* [ ] (an empty array)
* { } (an empty object)

## if empty return true and non empty return false
